# Images Folder

Place your static images here.
The application expects the following files:
- `hero-bg.jpg` (or webp) - High quality, landscape, moody black and white
- `gallery-preview.jpg` - Landscape, showing a photo gallery
- `workflow-wedding.jpg` - Portrait/Landscape, wedding photography B&W
- `testimonial-1.jpg`, `testimonial-2.jpg`, etc. - Portrait headshots B&W

## Search Terms for Google/Unsplash:
- "minimalist black and white photography"
- "luxury wedding photography black and white"
- "moody portrait photography black and white"
